Khai niem Flexbox:

- FlexContainer(the cha): la the chua
- FlexItem(the con): la cac the con
-- Main size: kich thuoc cua chieu song song voi main axis (truc hoanh) của các thẻ con
-- Cross size: kich thuoc cua chieu song song voi cross axis (truc tung) của các thẻ con
- Main axis: la truc hoanh
-- Diem dau cua main axis: la main start
-- Diem cuoi cua main axis: la main end
- Cross axis: la truc tung (vuong goc voi truc hoanh)
-- Diem dau cua cross axis: la cross start
-- Diem cuoi cua cross axis: la cross end

Nhung thuoc tinh cua Flexbox:

- Display(muon su dung flexbox phai co display): flex / inline-flex
- Flex-direction(Thay doi phuong huong cua main axis): row(hang ngang)(default) / column(hang doc)
- Flex-wrap(tuy chon cách xuong dong): nowrap(default) / wrap (xuong dong)/ warp-reverse (nhay len tren)
- Flex-basis: xet kich thuoc cua main size
- Justify-content(Căn flex-item trong truong hop main-sis nam ngang): flex-start / flex-end / center / space-between / space-around
- Justify-self(Giong nhu thang tren dung de xet cho flexitem): flex-start / flex-end / center
- Align-content(Giong voi Justify-content nhung xet theo cross-asis): flex-start / flex-end / center
- Align-self(Giong thang tren dung de xet cho flexitem): flex-start / flex-end / center
- Flex-grow(Tang kich thuoc cua main-size): number
- Flex-shrink(Giam kich thuoc cua main-size): number
- flex(Shorthand cua flex-basis, flex-grow, flex-shrink): number
- Order(Thu tu cua flexitem): number
- Flexflow (Shorthand cua flex-wrap,flex-direction)
